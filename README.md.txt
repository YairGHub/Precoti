# 📑 Sistema de Gestión de Facturas - PRECOTI
---

## 🛠️ Paso 1: Preparación (Solo se hace una vez)

1. **Instalar Python:** - Ve a [python.org](https://www.python.org/downloads/) y descarga la versión para Windows.
   - **⚠️ MUY IMPORTANTE:** Al abrir el instalador, marca la casilla que dice **"Add Python to PATH"** antes de darle a "Install Now". Si no haces esto, el programa no arrancará.

2. **Descargar este Proyecto:**
   - Haz clic en el botón verde **"Code"** (arriba a la derecha en esta página) y elige **"Download ZIP"**.
   - Descomprime el archivo en una carpeta de tu computadora (ejemplo: en tu Escritorio).

---

## 🔑 Paso 2: Configuración de tu Llave de la API

1. Entra a la carpeta del proyecto y busca el archivo llamado '.env.example'.
2. Cámbiale el nombre: bórrale el '.example' para que se llame solo **'.env'**.
3. Haz clic derecho sobre el archivo '.env' -> **Abrir con...** -> **Bloc de Notas**.
4. Donde dice 'GEMINI_API_KEY=TU_CLAVE_AQUI', borra 'TU_CLAVE_AQUI' y pega tu clave real de Google.
5. Guarda el archivo (Ctrl + G) y ciérralo.

---

## 🚀 Paso 3: Cómo encender el programa

1. **Instalar librerías (Solo la primera vez):**
   - Abre la carpeta del proyecto.
   - Haz clic en la barra de direcciones de la carpeta (arriba, donde sale la ruta), borra todo, escribe 'cmd' y presiona **Enter**.
   - En la ventana negra que aparece, escribe: 'pip install -r requirements.txt' y presiona Enter. Espera a que termine.

2. **¡A correr!:**
   - Cada vez que quieras usar el programa, simplemente haz **doble clic** en el archivo:
     👉 **'INICIAR_PROYECTO.bat'**
   - Se abrirán dos ventanas negras y, automáticamente, tu navegador mostrará el sistema listo para usar. 
   - *Nota: No cierres las ventanas negras mientras estés usando el programa.*
